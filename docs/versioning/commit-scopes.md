# Commit Scopes

Use one of these scopes in commit messages:
- `web`
- `api`
- `db`
- `infra`
- `docs`
- `ci`
- `ops`

Format:
`<type>(<scope>): <summary>`

Examples:
- `feat(api): add payment verification endpoint`
- `fix(web): align hero image on desktop`
- `chore(ci): add pull request build workflow`
