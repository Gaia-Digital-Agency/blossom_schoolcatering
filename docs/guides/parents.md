# Parent User Guide

## Login
- Open `/schoolcatering/parent/login`
- Sign in with parent credentials

## Main Parent Page
- Open `/schoolcatering/parents`
- View linked youngsters
- View allergy snapshot (auto-filled from youngster profile)
- Select date/session and browse menu

## Ordering
- Create cart per youngster/date/session
- Add up to 5 items per cart/order
- Submit order before cutoff
- Edit or delete before cutoff window closes

## Useful Features
- Consolidated orders view
- Quick reorder
- Meal plan wizard
- Favourite meal combos
- Spending dashboard
- Billing consolidated page
- Upload proof of payment and download receipt after verification

## Rules
- Weekend and blackout order blocks are enforced
- Session availability (Snack/Breakfast) follows admin settings
- Lunch is always enabled
