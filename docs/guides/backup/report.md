# Report User Guide

Last updated: 2026-03-20

## Access
- Route: `/schoolcatering/admin/reports`
- Role: `ADMIN`

## Report Scope
- Orders
- Revenue
- Billing
- School and delivery filtering

## Typical Filters
- Date range
- School
- Parent
- Delivery user
- Session
- Dish
- Order status
- Billing status

## Usage Tip
- Use broad date ranges first, then narrow by school or session to investigate specific issues.
