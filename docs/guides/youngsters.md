# Youngster User Guide

Last updated: 2026-02-28

## Access
- Login: `/schoolcatering/youngster/login`
- Main page: `/schoolcatering/youngsters`

## Core Modules
- Confirmed order of the day.
- Weekly nutrition and badge insight.
- Menu and draft cart ordering.

## Insights Panel
- Badge level (`NONE`, `BRONZE`, `SILVER`, `GOLD`, `PLATINUM`).
- Consecutive order metrics.
- Current month order count.
- Weekly calories/order summary.
- Birthday countdown.

## Ordering Flow
1. Select service date and session.
2. Add menu items to draft (max 5 distinct items).
3. Adjust quantities/remove items.
4. Place order.

## Rules
- Cutoff countdown is shown and enforced.
- Session availability depends on admin session settings.
- Ordering on blocked dates/times is rejected by API.

## Profile Edit Note
- Youngster page is operational; profile corrections are handled by Admin modules.
