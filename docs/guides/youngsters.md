# Youngster User Guide

## Access
- Login: `/schoolcatering/youngster/login`
- Main page: `/schoolcatering/youngsters`

## Core Modules
- Weekly Nutrition + Badge insights
- Session menu + draft cart
- Registration details update shortcut

## Insights
- Badge level: `NONE` / `BRONZE` / `SILVER` / `GOLD` / `PLATINUM`
- Max consecutive order days/weeks
- Current month orders
- Weekly calories, order count, and dish count

## Ordering Flow
- Pick `Service Date` and `Session`
- Menu is loaded from active admin dishes
- Allergies field is shown read-only
- Add dishes to draft (max 5 items)
- Adjust quantity or remove dish
- Place order

## Draft and Cutoff
- Open draft carts are auto-loaded
- Cutoff countdown is shown on page
- Ordering after cutoff is blocked

## Notes
- `Update Registration Details` goes to `/register/youngsters`
- Non-lunch sessions may be unavailable when disabled by admin
