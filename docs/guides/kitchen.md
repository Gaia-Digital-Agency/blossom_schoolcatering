# Kitchen User Guide

## Login
- Open `/schoolcatering/kitchen/login`
- Sign in with kitchen credentials

## Pages
- `/schoolcatering/kitchen` and `/schoolcatering/kitchen/today`
- `/schoolcatering/kitchen/yesterday`
- `/schoolcatering/kitchen/tomorrow`

## Top Controls
- `Yesterday`, `Today`, `Tomorrow`, `Refresh Now` are available in one row
- Use `Refresh Now` for immediate reload

## Today View
- `Overview` table:
- `Total Orders | Total Dishes | Lunch | Snack | Breakfast`
- `Summary` table:
- grouped by dish name with quantity
- `Allergen Alerts` section
- `Orders` board with two columns:
- left = pending orders
- right = completed orders
- click an order card to move it between columns

## Yesterday and Tomorrow Views
- Show only:
- `Overview` table
- `Summary` table
- Do not show:
- `Allergen Alerts`
- `Orders` board

## Refresh Behavior
- Auto-refresh every 60 minutes
- Active between `05:00` and `21:00` (Asia/Makassar)
- Manual refresh is always available
