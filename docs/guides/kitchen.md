# Kitchen User Guide

## Login
- Open `/schoolcatering/kitchen/login`
- Sign in with kitchen credentials

## Kitchen Pages
- `/schoolcatering/kitchen` (today dashboard)
- `/schoolcatering/kitchen/yesterday`
- `/schoolcatering/kitchen/today`
- `/schoolcatering/kitchen/tomorrow`

## What You See
- Daily total orders and total dishes
- Session breakdown (Breakfast, Snack, Lunch)
- Allergen alerts
- Order preparation list

## Refresh Behavior
- Auto-refresh every 60 minutes
- Active between 05:00 and 21:00 (Asia/Makassar)
- Manual refresh button is available
