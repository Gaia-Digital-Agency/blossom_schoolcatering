# Kitchen User Guide

## Access
- Login: `/schoolcatering/kitchen/login`
- Routes:
- `/schoolcatering/kitchen` (today)
- `/schoolcatering/kitchen/today`
- `/schoolcatering/kitchen/yesterday`
- `/schoolcatering/kitchen/tomorrow`

## Top Controls
- `Yesterday`, `Today`, `Tomorrow`, `Refresh Now`
- Date tab changes summary context immediately

## Today View
- Overview table (`Total Orders`, `Total Dishes`, `Lunch`, `Snack`, `Breakfast`)
- Dish summary table
- Allergen alerts section
- Orders board (pending/completed columns)

## Yesterday and Tomorrow
- Overview + summary only
- Order board and allergen alert board are hidden

## Refresh Behavior
- Manual refresh anytime
- Auto-refresh schedule can run during operational window
