# Kitchen User Guide

Last updated: 2026-02-28

## Access
- Login: `/schoolcatering/kitchen/login`
- Routes:
  - `/schoolcatering/kitchen` (today)
  - `/schoolcatering/kitchen/today`
  - `/schoolcatering/kitchen/yesterday`
  - `/schoolcatering/kitchen/tomorrow`

## Main Controls
- `Yesterday`, `Today`, `Tomorrow`
- `Refresh Now`

## Today View
- Overview metrics (`Total Orders`, `Total Dishes`, per-session counts).
- Dish summary table.
- Dietary alert cards.
- Order boards (`Pending`, `Completed`).
- Pending cards can be marked with `Mark Kitchen Complete`.

## Yesterday/Tomorrow Views
- Summary-focused view.
- No active production action board.

## Refresh Behavior
- Manual refresh available anytime.
- Automatic refresh runs hourly during 05:00-21:00 (Asia/Makassar).
